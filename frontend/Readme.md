## BSC NFTs Rarity

Synthetix block explorer.

### Run

- Copy `.env.local.sample` to `.env.local` and configure.
- Run `make` to start the app.
- Visit the dapp at http://localhost:1313

## Stack

- [React](https://reactjs.org/)
- [Material UI](https://material-ui.com/)
